
Design patterns are categorized into three main types based on their purpose:

Creational Patterns:
- Singleton Pattern
- Factory Method Pattern
- Abstract Factory Pattern
- Builder Pattern
- Prototype Pattern

Structural Patterns:
- Adapter Pattern
- Decorator Pattern
- Facade Pattern
- Proxy Pattern
- Composite Pattern
- Bridge Pattern
- Flyweight Pattern

Behavioral Patterns:
- Observer Pattern
- Strategy Pattern
- Command Pattern
- Chain of Responsibility Pattern
- Template Method Pattern
- State Pattern
- Visitor Pattern
- Interpreter Pattern
- Memento Pattern
- Mediator Pattern
